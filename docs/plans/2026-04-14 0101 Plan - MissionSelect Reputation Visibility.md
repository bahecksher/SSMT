# Plan - MissionSelect Reputation Visibility
_Created: 2026-04-14 0101_

## Goal
Restore reputation visibility on MissionSelect so players can see both mission rep gains and current company standing at a glance.

## Approach
- Update mission cards to show rep payout alongside the existing credit reward.
- Reintroduce standing details on favor cards using saved company rep and standing thresholds.
- Keep the layout compact enough to survive narrow mobile viewports, using a minimal progress indicator instead of a heavier detail panel.

## Scope boundaries
- No changes to mission generation, rep awards, or favor economy values
- No changes to corporation selector behavior on the main menu
- No new screens, modals, or company detail panels

## Open questions
- Whether the favor standing display should eventually become more explicit still depends on live phone playtesting.
