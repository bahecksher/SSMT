# Spec - MissionSelect Reputation Visibility
_Created: 2026-04-14 0101_

## Goal
Make MissionSelect clearer about reputation progression by showing what rep a mission pays out and how much standing the player currently has with each corporation.

## Mission card changes
- Each mission card should show the mission's credit reward and its rep payout.
- Rep payout should be visible without opening another screen or relying on memory of mission tiers.
- Existing mission brief copy should stay visible.

## Favor card changes
- Each favor card should show the corporation's current standing label and raw rep total.
- Favor cards should indicate progress toward the next standing tier when one exists.
- The existing favor offer and cost behavior stays the same.

## Out of scope
- Rebalancing mission rep rewards
- Reworking favor costs or offers
- Changing manual corporation selection behavior
